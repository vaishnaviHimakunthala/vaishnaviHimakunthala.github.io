## Writing a new post

Adding a post is easy — just drop a new markdown file in `src/blog/`:

```markdown
---
title: My next post
description: A one-line summary shown in previews.
date: 2026-08-01
tags: [projects]
---

Your content here, in plain markdown.
```

Set `draft: true` in the frontmatter to keep a post hidden while you work on it.

That's it — the blog index and homepage pick it up automatically.
